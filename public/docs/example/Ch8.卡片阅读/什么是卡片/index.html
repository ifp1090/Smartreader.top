    什么是卡片？

    在了解卡片之前不妨做一个游戏吧！

    每个人都能记住自己的电话号码，这看起来没什么奇怪的。可是，如果我说你不一定记的住你的电话号码，你相信吗？

    如果你不相信，请你跟着我的顺序做一遍，看看你是不是真的不一定记得住：）

    第一步，说出你的手机号码

    你会发现，你刚刚说出来的号码，可能遵循了类似格式：

    • XXX-XXXX-XXXX
    • XXXX-XXXX-XXX

    也就是说，你可能不自主尝试将你的电话号码进行了“分组”，那么这时我们进行第二步：

    你能快速用一下“分组法”脱口而出你的电话号码吗？

    • XX－XXXX－XX-XX

    注意，你需要“脱口而出”哦！

    你会发现，相比较第一步的快速复述，这时你可能要停顿一下甚至还可能停顿几秒后才能说出你的电话号码。

    这是为什么？

    为什么我们可能复述超过1000次甚至10000次以上的手机号，只要稍微改一下复述方式，就大概率“失忆”了呢？

    答案很简单，看起来我们记忆的是自己的手机号，其实我们记忆的是几个≤4的“数字串”。

    这其实和我们大脑的记忆原理息息相关。

    首先我们看一下我们对11位号码的分组：

    • XXX-XXXX-XXXX
    • XXXX-XXXX-XXX

    上述分组有什么特点呢？

    第一，分组的组数≤4

    不论我们按照3－4－4或者4－4－2的组合方式，我们将它进行分组的组数也≤4，也就是说，一个≤4的“组数”，我们大脑也可以比较轻松的记住它。

    反过来思考，假如我们的电话一共有27位，即便我们将它转成了这样的形式：

    XXX－XXXX－XXXX－XXXX－XXXX－XXXX－XXXX

    这样我们也无法轻松记住，依然要借助其他的工具来帮我们记忆，而一旦要借助第三方工具来记忆，又造成了大脑的行动负荷：我们每次都要打开第三方工具（这可能需要一连串的步骤，比如拿起手机 － 点击某个APP － 点击某个栏目或者搜索输入关键词 － 点击查询……)，这一连串动作对大脑而言会比较复杂，所以很可能陷入行为瘫痪。

    第二，分组后的数字≤4

    我们将11位数字分组后，最终每个组呈现的是≤4的限额，相比一长串数字，我们的大脑更容易记住≤4的“数字组”。

    反过来思考，如果我们的电话号码保持11位不变，如果我们将其分成这样的形式：

    XXXXXX－XXXXX

    我们的大脑也无法轻松记住它们。

    所以，一旦我们将一个超过大脑记忆区间的数字用分组的方式，就能不依赖于任何第三方工具，仅仅通过自己的大脑就能快速回忆起来，而一旦快速回忆起来，就能灵活运用到我们需要它的地方。

    这就是我们大脑在记忆层面神奇的地方！

    这时候，我们把这个案例中的“电话号码”换成一本书，就能很容易明白，为什么我们读了很多书，但是好像记不住书中的知识了 —— 因为书中的字数已经远远超出了我们大脑所能快速记住并回忆的范畴！

    但是，我们刚刚说过，我们能记住自己的电话号码，并不是记忆的11位数字，而是3个组块。所以，对于一本书，我们要记住的也不是它的字数，而是我们自定义的“组块”。也就是说，如果我们将书中的文字根据某种方式进行“分组”最后成为了“知识组块”，那么我们就可以更容易的记住它们。

    那么，我们如何对一本书的知识进行“组块”呢？

    这就回到了我们开头所介绍的知识 —— 卡片。

    换句话说，类似于将11位电话号码进行分组一样，一旦分组后，我们就能将这个电话号码轻松用在各种地方 —— 注册、登记、交友、接收信息等等，换句话说，这些分组的数字被我们不断复用，产生价值。

    同样，我们也可以将一本书中的重点知识组合成卡片，这样我们去读一本书，就不再是读这些文字本身，而是将它们组合成一张又一张能够被我们复用的卡片。

    就能不断的运用这些卡片来帮助我们对知识进行更深入的理解与运用。

    那么，

    电话号码是11位，它也超出了我们大脑的记忆范畴，但我们依然是可以记住它的！因为我们将它进行了满足两个“≤4”的进分组。换句话说，我们将超过我们大脑记忆区间的知识，通过分组的方式变成了“知识组块”，这样就给我们带来了两大便利：

    第一，不用花太多力气就能记住并且也很容易回忆起来（想想你轻松复述11位的电话号码）

    第二，一旦记得住并且容易回忆，就能大幅度增加用上它的可能性（想想你可以轻松从大脑中回忆起11位的电话号码来给到谈得来的朋友）

    回到一本书的阅读，我们如何将书中的知识记住并运用呢？
